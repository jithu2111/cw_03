Exercise 2: Debugging CSS Code

Find and fix 3 common CSS mistakes in the provided code.

Demonstration Files: buggy.css | corrected.css

Identified Mistakes:

  1. Mistake 1: Missing semicolon after color: red in the body rule.
    Fix: Add semicolon: color: red;
  2. Mistake 2: The font-family property lacks fallback fonts.
    Fix: Add fallback fonts: font-family: "Times New Roman", Times, serif;
  3. Mistake 3: Missing semicolon after box-sizing: border-box in the .container rule.
    Fix: Add semicolon: box-sizing: border-box;

Corrected CSS Code:

/* Corrected CSS */
body {
    font-family: "Times New Roman", Times, serif;
    margin-left: 20%;
    margin-right: 20%;
    text-align: justify;
    background: ivory;
    color: red;
}

h1, h2, h3 {
    color: #ff0000;
    font-size: 2.5em;
    margin: 20px 0px 20px 0px;
}

.container {
    width: 100%;
    padding: 20px;
    box-sizing: border-box;
}

a {
    color: #0000FF;
    text-decoration: underline;
}

Exercise 4: External Stylesheet Creation

External Stylesheet Implementation

An external stylesheet lyrics.css has been created with the following classes:

  • .artist: 14pt, Arial, Bold, Moccasin
  • .song-title: 14pt, Arial, Normal, Khaki
  • .album: 12pt, Arial, Normal, Khaki
  • .year: 12pt, Arial, Bold, Ivory
  • .verse: 14pt, Trebuchet MS, Bold, Gold, margin-left 2cm, margin-right 2cm
  • .chorus: 14pt, Trebuchet MS, Bold & Italic, Gold, margin-left 4cm, margin-right 4cm

To use this stylesheet, link it in the HTML: <link rel="stylesheet" href="lyrics.css">
